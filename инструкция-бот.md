# 🎂 Telegram-бот для напоминания о днях рождениях

Бот, который будет напоминать вам и вашим коллегам о днях рождениях в общем чате.

## 🚀 Быстрый старт

### 1. Получение токена бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Придумайте имя для бота (например, `Birthday Reminder Bot`)
4. Придумайте юзернейм для бота (должен заканчиваться на `bot`, например `birthday_reminder_bot`)
5. BotFather выдаст вам токен — скопируйте его

### 2. Настройка бота

1. Откройте файл `config.py`
2. Замените `YOUR_BOT_TOKEN_HERE` на полученный токен:

```python
BOT_TOKEN = "1234567890:ABCdefGHIjklMNOpqrsTUVwxyz"
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Запуск бота

```bash
python bot.py
```

## 📋 Использование

### Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и основная информация |
| `/help` | Подробная инструкция |
| `/add` | Добавить коллегу в список |
| `/list` | Показать всех коллег с датами рождения |
| `/delete` | Удалить коллегу из списка |
| `/setup_chat` | Настроить текущий чат для уведомлений |

### Добавление в групповой чат

1. Добавьте бота в групповой чат Telegram
2. Дайте боту права на отправку сообщений
3. Отправьте команду `/setup_chat` в чате
4. Добавьте коллег командой `/add` (можно делать в личном чате с ботом)

### Пример добавления коллеги

```
Вы: /add
Бот: Введите имя коллеги:
Вы: Иван Петров
Бот: Теперь введите дату рождения в формате ДД.ММ (например, 15.05):
Вы: 15.05
Бот: 🎉 Коллега Иван Петров добавлен! День рождения: 15.05
```

## ⏰ Как работают уведомления

- Бот проверяет дни рождения каждый день в **09:00 UTC**
- Если у кого-то из коллег день рождения, бот отправит сообщение в настроенный чат
- Формат сообщения: `🎉 Сегодня день рождения у [Имя]! Поздравляем! 🎂🎈`

### Изменение времени уведомлений

Откройте `config.py` и измените параметр `NOTIFICATION_TIME`:

```python
# Время в формате ЧЧ:ММ по UTC
NOTIFICATION_TIME = "09:00"  # 09:00 UTC
```

## 🔧 Настройка для разных часовых поясов

По умолчанию бот использует время UTC. Если вы хотите изменить время уведомлений под свой часовой пояс, пересчитайте время:

- Москва (UTC+3): `09:00` UTC = `12:00` по Москве → ставим `09:00` в конфиге
- Алматы (UTC+5): `09:00` UTC = `14:00` по Алматы → ставим `09:00` в конфиге

Проще всего указать желаемое время по UTC на сайте [time.is](https://time.is/UTC)

## 🛠 Запуск в фоновом режиме

### Linux/MacOS

```bash
nohup python bot.py &
```

### Использование systemd (Linux)

Создайте файл `/etc/systemd/system/birthday-bot.service`:

```ini
[Unit]
Description=Telegram Birthday Bot
After=network.target

[Service]
Type=simple
User=your_user
WorkingDirectory=/path/to/birthday_bot
ExecStart=/usr/bin/python3 /path/to/birthday_bot/bot.py
Restart=always

[Install]
WantedBy=multi-user.target
```

Запустите:

```bash
sudo systemctl enable birthday-bot
sudo systemctl start birthday-bot
```

### Windows

Используйте Task Scheduler или запустите как службу через NSSM.

## 🐛 Решение проблем

**Бот не отправляет уведомления:**
- Проверьте, что бот добавлен в чат и имеет права на отправку сообщений
- Убедитесь, что команда `/setup_chat` была выполнена в целевом чате
- Проверьте логи на наличие ошибок

**Бот не запускается:**
- Убедитесь, что токен в `config.py` указан правильно
- Проверьте, что установлены все зависимости: `pip install -r requirements.txt`
