## 📁 Как связаны файлы вашего проекта

```
birthday_bot/
│
├── config.py         ← Настройки (токен, время уведомлений)
│      │
│      └── bot.py читает отсюда BOT_TOKEN и NOTIFICATION_TIME
│
├── database.py       ← Работа с базой данных (SQLite)
│      │
│      └── bot.py вызывает функции: add_colleague(), get_all_colleagues()
│
├── bot.py            ← Главный файл! Запускается и управляет всем
│      │
│      ├── Подключается к Telegram через токен
│      ├── Читает команды (/start, /add, /list...)
│      ├── Вызывает функции из database.py
│      └── Планировщик (APS) будит бота в 9:00 для уведомлений
│
├── birthdays.db      ← База данных (создаётся автоматически)
│      │
│      └── Таблицы: colleagues, chat_settings
│
└── requirements.txt  ← Список библиотек для установки
```

## 🔄 Что происходит, когда вы запускаете бота

### Шаг 1: Запуск
```bash
python bot.py
```

1. `bot.py` загружается
2. Импортирует `BOT_TOKEN` из `config.py`
3. Создаёт объект `Database()` → создаётся файл `birthdays.db`
4. Бот подключается к серверам Telegram

### Шаг 2: Бот ждёт сообщения
```
[Бот запущен и слушает сообщения]
```

### Шаг 3: Вы пишете `/start` в Telegram

```
Telegram сервера → bot.py (функция start_command) → Ответ вам
```

Бот отвечает: *"Привет! Я бот для напоминания о днях рождениях..."*

### Шаг 4: Вы пишете `/add`

```
1. Бот просит: "Введите имя коллеги"
2. Вы: "Иван"
3. Бот просит: "Введите дату рождения"
4. Вы: "15.05"
5. Бот вызывает: db.add_colleague("Иван", "15.05")
6. database.py записывает в birthdays.db
7. Бот отвечает: "✅ Коллега Иван добавлен!"
```

### Шаг 5: Уведомление в 9:00 UTC

```
[Планировщик APS срабатывает в 9:00]
     │
     ▼
send_birthday_notifications(bot)
     │
     ▼
db.get_colleagues_by_date(месяц=текущий, день=текущий)
     │
     ▼
Если есть именинники → bot.send_message(CHAT_ID, "🎉 Сегодня день рождения у...")
```

## 🛠️ Как запустить у себя на ПК (для отладки)

### 1. Установите Python (если нет)
```bash
python --version
```

### 2. Создайте виртуальное окружение ⭐

**Виртуальное окружение** — это изолированная среда для проекта. Все библиотеки устанавливаются в отдельную папку `venv/`, а не в систему.

**Преимущества:**
- Нет конфликта версий с другими проектами
- Легко перенести на другой компьютер
- Чистая система без лишних библиотек
- Можно указать точные версии библиотек

**Создание:**
```bash
python -m venv venv
```

Появится папка `venv/` — это и есть виртуальное окружение.

**Активация:**

| ОС | Команда |
|----|---------|
| **Windows (PowerShell)** | `.\venv\Scripts\Activate.ps1` |
| **Windows (cmd)** | `venv\Scripts\activate.bat` |
| **Linux/Mac** | `source venv/bin/activate` |

После активации в начале строки появится `(venv)`:
```bash
(venv) user@pc:~/birthday_bot$
```

**Деактивация** (когда закончили работу):
```bash
deactivate
```

### 3. Установите библиотеки
```bash
pip install -r requirements.txt
```

### 4. Вставьте токен бота

1. В Telegram найдите [@BotFather](https://t.me/BotFather)
2. Напишите `/newbot`
3. Придумайте имя и username
4. BotFather даст токен (например: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)
5. Вставьте в `config.py`:
   ```python
   BOT_TOKEN = "123456789:ABCdefGHIjklMNOpqrsTUVwxyz"
   ```

### 4. Запустите
```bash
python bot.py
```

Увидите в консоли:
```
2026-03-15 10:00:00 - bot - INFO - Бот запущен
2026-03-15 10:00:00 - bot - INFO - Планировщик настроен на 09:00 UTC
```

### 5. Проверьте в Telegram
Найдите своего бота по username, напишите `/start`

## 🖥️ Чем отличается запуск на ПК и на VPS

| Аспект | Локальный ПК | VPS (сервер) |
|--------|--------------|--------------|
| **Доступ** | Только когда ПК включен | 24/7 |
| **Интернет** | Ваш домашний | Стабильный, быстрый |
| **ОС** | Windows/Mac/Linux | Обычно Linux (Ubuntu) |
| **Запуск** | Вручную `python bot.py` | Автозапуск при загрузке |
| **Токен** | В коде (небезопасно) | В переменных окружения |

## 📦 Что происходит при переезде на VPS

1. **Копируете файлы** на сервер:
   ```
   bot.py, config.py, database.py, requirements.txt
   ```

2. **На VPS** (Ubuntu):
   ```bash
   # Установить Python и библиотеки
   sudo apt update
   sudo apt install python3 python3-pip
   pip3 install -r requirements.txt

   # Вставить токен в config.py

   # Запустить
   python3 bot.py &
   ```

3. **Бот работает**, даже если вы закрыли терминал
